# twitter-integration
Serviço responsável por prover uma interface de acesso simplificada a API do Twitter para coleta de dados de usuários e suas publicações na rede social
___

### Desenvolvimento
Esse app é composto por
- Express.js
- RabbitMQ
- MongoDB
- Jest

### Testes
Executar 
> npm run test

### Execução
Executar
> npm run start
